# 🤖 SmartChat AI

[![Node.js](https://img.shields.io/badge/Node.js-22.x-green)](https://nodejs.org/)

An enterprise-grade, production-ready AI chatbot powered by **Groq Llama 3.3 API** with real-time streaming responses, voice input, MongoDB persistence, JWT authentication, and rate limiting.
